# 3 DOF Arm Manipulator

## Overview
This project implements a 3 Degree of Freedom robotic arm manipulator designed for basic object manipulation and pick-and-place tasks.

## Hardware Used
- Servo Motors
- Microcontroller (ESP32 / Arduino)
- Servo Driver (PCA9685)
- Power Supply
- Mechanical Arm Structure

## Software
- Arduino IDE
- C++ Programming

## Features
- 3 DOF movement
- Servo based control
- Forward kinematics implementation
- Pick and place operation

## Applications
- Educational robotics
- Industrial automation concepts
- Robot arm control experiments
